# 🏎️ Analýza Formule 1 pomocí SQL

V tomto projektu jsem analyzovala historická data F1 (1950-2024). Cílem bylo procvičit si SQL dotazy, propojování tabulek (JOIN) a čištění dat.

## 📊 Co jsem zjistila?
* **Dominance:** Nejsilnější dvojicí historie je Lewis Hamilton s Mercedesem (82 výher).
* **Rekordy:** Michael Schumacher drží prvenství v počtu nejrychlejších kol.
* **Čištění dat:** V datasetu bylo přes 1000 neúplných záznamů, které jsem pomocí SQL odfiltrovala.

## 🛠️ Použité nástroje
* **SQL (SQLite)**
* **DBeaver** pro správu databáze
